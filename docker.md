# Docker

## Command
* docker images
* dcoker ps
* docker run [option] imageName
  * -d : background 
  * --name : container name
  * -p port:port : portforwarding
  * -i : Keep STDIN open even if not attached
  * -t : Allocate a pseudo-tty
  * -rm : remove container if container is terminated
* docker rm container
* docker rmi image
* docker pull id/image:tag
